# Attività di rilascio di Workfront Fusion: settimana del 9 febbraio 2026

Questa pagina descrive tutti i miglioramenti apportati ad Adobe Workfront Fusion la settimana del 9 febbraio 2026.

Per un elenco di tutte le modifiche recenti, consulta [Attività sulla versione di Adobe Workfront Fusion](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

Per un elenco delle correzioni di bug recenti in Workfront Fusion, consulta la pagina [Aggiornamenti di manutenzione di Workfront](https://experienceleague.adobe.com/it/docs/workfront-known-issues/releases/current-updates) e controlla la presenza di eventuali aggiornamenti etichettati Aggiornamento di manutenzione di Workfront Fusion.

## Aggiornamento a guardrail di esecuzione incompleti

Per supportare meglio scenari grandi e complessi, abbiamo aggiornato i guardrail per le esecuzioni incomplete. Ora i limiti di esecuzione incompleti si riferiscono allo scenario. Fusion può memorizzare fino a 1 GB o 100 esecuzioni incomplete per scenario, a seconda del limite raggiunto per primo.

In precedenza, i limiti erano basati sui team.

Per ulteriori informazioni sui guardrail in Fusion, vedere [Guardrail delle prestazioni di Fusion](/help/workfront-fusion/references/scenarios/fusion-performance-guardrails.md).

## Nuovo Microsoft SharePoint Online > Crea un modulo file

Al connettore Microsoft SharePoint Online viene aggiunto il nuovo modulo Create a file. Il nuovo modulo offre prestazioni migliori quando si lavora con file di grandi dimensioni.

Il modulo Create a file precedentemente disponibile è stato rinominato Create a file (Legacy). Anche se il modulo legacy è ancora disponibile, si consiglia di utilizzare il nuovo modulo.

Per ulteriori informazioni sui moduli SharePoint Online di Microsoft, vedere [Moduli SharePoint Online di Microsoft](/help/workfront-fusion/references/apps-and-modules/third-party-connectors/sharepoint-modules.md).
