# Attività sulla versione di Workfront Fusion: settimana del martedì 16 febbraio 2026

Questa pagina descrive tutti i miglioramenti apportati a Adobe Workfront Fusion la settimana del martedì 16 febbraio 2026.

Per un elenco di tutte le modifiche recenti, consulta [Attività sulla versione di Adobe Workfront Fusion](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

Per un elenco delle correzioni di bug recenti in Workfront Fusion, consulta la pagina [Aggiornamenti di manutenzione di Workfront](https://experienceleague.adobe.com/it/docs/workfront-known-issues/releases/current-updates) e controlla la presenza di eventuali aggiornamenti etichettati Aggiornamento di manutenzione di Workfront Fusion.

## Aggiungere prompt di IA agli scenari con il nuovo modulo dell’agente MCP

È possibile includere un prompt di IA nello scenario utilizzando il protocollo MCP (Model Context Protocol) combinato con i modelli di linguaggio di grandi dimensioni (LLM, Large Language Model). Configurandoli nel modulo dell’agente MCP, puoi usare l’intelligenza artificiale per impostare flussi di lavoro efficienti, sicuri e flessibili.

MCP consente di utilizzare un prompt di IA per la connessione con le applicazioni. Poiché il prompt può tenere conto di varianti e contesto, questo riduce la necessità di configurare manualmente alcuni flussi di lavoro, route o sezioni di scenario in Fusion.

Per ulteriori informazioni sull&#39;utilizzo dei prompt di IA in Fusion, vedere [Aggiungere un prompt di IA allo scenario](/help/workfront-fusion/create-scenarios/add-modules/add-an-ai-prompt-to-your-scenario.md).

Per informazioni sulla configurazione del modulo dell&#39;agente MCP, vedere [Modulo dell&#39;agente MCP](/help/workfront-fusion/references/apps-and-modules/tools-and-transformers/model-context-protocol-mcp-connector.md).

## API operazioni ora disponibili

Abbiamo reso possibile ottenere i dati delle operazioni tramite l’API di Workfront Fusion. Ora è possibile recuperare dati operativi dettagliati e conteggi delle operazioni per le organizzazioni e raggruppare e filtrare i risultati in base a diversi parametri.

Per ulteriori informazioni, vedere [API Workfront Fusion](https://developer.adobe.com/workfront-fusion-apis/#)

## Visualizzare le metriche delle prestazioni a livello di organizzazione nel dashboard delle prestazioni di Fusion

Per semplificare la comprensione delle prestazioni di Fusion all&#39;interno dell&#39;organizzazione, è stata creata la nuova dashboard delle prestazioni. Ora gli amministratori di Fusion possono vedere rapidamente quali scenari sono più in esecuzione, dove si verificano ritardi e con quale efficacia funzionano i pool di lavoro. Il dashboard delle prestazioni fornisce visibilità in tempo reale sui volumi di esecuzione, sulla profondità della coda, sull&#39;utilizzo del pool e sulle prestazioni a livello di scenario.

Per informazioni sul dashboard delle prestazioni di Fusion, vedere [Visualizzare il dashboard delle prestazioni per un&#39;organizzazione](/help/workfront-fusion/set-up-and-manage-workfront-fusion/set-up-and-manage-orgs-and-teams/set-up-orgs-teams-and-users/view-performance-dashboard.md).
