## Rendi disponibile un modulo di chiamata API personalizzata (polling) nel connettore Adobe Authenticator

Per consentire la ripetizione delle chiamate API fino al raggiungimento di una determinata condizione o limite, è stato creato il modulo Fai una chiamata API personalizzata (Polling). Questo modulo include opzioni che consentono di configurare le chiamate API ripetute. Puoi far eseguire la chiamata API fino a quando non viene soddisfatta una particolare condizione, impostare un numero specifico di tentativi e configurare l’intervallo di tempo tra i tentativi.

Per ulteriori informazioni, consulta [Effettuare una chiamata API personalizzata (polling)](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/adobe-authenticator-modules.md#make-a-custom-api-call-polling) nell&#39;articolo Moduli di Adobe Authenticator.
