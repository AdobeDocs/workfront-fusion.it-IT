# Attività di rilascio di Workfront Fusion: settimana del 17 novembre 2025

Questa pagina descrive tutti i miglioramenti apportati ad Adobe Workfront Fusion la settimana del 17 novembre 2025.

Per un elenco di tutte le modifiche recenti, vedere [Attività di Adobe Workfront Fusion release](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

Per un elenco delle correzioni di bug recenti in Workfront Fusion, vedere la pagina [Aggiornamenti di manutenzione di Workfront](https://experienceleague.adobe.com/en/docs/workfront-known-issues/releases/current-updates) e verificare la presenza di eventuali aggiornamenti etichettati Aggiornamento di manutenzione di Workfront Fusion.

## È ora disponibile un nuovo connettore Slack

Per espandere le funzionalità del connettore Slack e mantenerlo aggiornato con gli aggiornamenti apportati all’applicazione Slack, abbiamo creato un nuovo connettore Slack. Con il nuovo connettore è possibile:

* Creare, leggere, aggiornare o eliminare messaggi e file
* Crea, unisci, lascia, archivia, annulla archiviazione o recupera informazioni dai canali
* Usa pin, stelle, reazioni e elementi salvati
* Invitare, elencare, recuperare o richiamare utenti
* Creare promemoria e stati
* Avvia uno scenario basato su nuovi messaggi, utenti, eventi o file
* Effettuare una chiamata personalizzata all’API Slack

Per informazioni sul nuovo connettore Slack, vedere [Moduli Slack](/help/workfront-fusion/references/apps-and-modules/third-party-connectors/slack-modules-new.md).

## Connettore Adobe Substance ora disponibile

Per semplificare l’utilizzo di Adobe Substance con Fusion, è stato creato un connettore Adobe Substance dedicato. Con i moduli Adobe Substance puoi:

* Creare, eseguire il rendering o recuperare dati su una scena 3D
* Conversione di file 3D
* Genera un composito
* Creare uno spazio

Per informazioni, vedere [Moduli di Adobe Substance](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/adobe-substance-modules.md).

## Il modulo Adobe Photoshop > Applica modifiche PSD ora supporta i file di grandi dimensioni

Per semplificare la modifica di file Photoshop di grandi dimensioni, il modulo Adobe Photoshop > Applica modifiche PSD ora supporta i file di grandi dimensioni.

Anche se non esiste un limite di dimensione definito per il file per la piattaforma Workfront Fusion, il timeout dello scenario o altri fattori possono influire sull’elaborazione di file di grandi dimensioni.

Per ulteriori informazioni sui file di grandi dimensioni, vedere [Utilizzo dei file di grandi dimensioni](/help/workfront-fusion/references/scenarios/fusion-large-files.md).

Per ulteriori informazioni sui moduli Adobe Photoshop, vedi [Moduli Adobe Photoshop](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/adobe-photoshop-modules.md).
