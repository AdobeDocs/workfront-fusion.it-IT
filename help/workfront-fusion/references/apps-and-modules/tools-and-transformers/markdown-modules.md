# [!UICONTROL Moduli Markdown]

Per informazioni sui moduli, consulta gli articoli in [Moduli: indice degli articoli](/help/workfront-fusion/references/modules/modules-toc.md).

## [!UICONTROL Da HTML a Markdown]

Questo modulo converte il codice HTML in Markdown.

<table style="table-layout:auto"> 
 <col> 
 <col> 
 <tbody> 
  <tr> 
   <td role="rowheader">[!UICONTROL Markdown]</td> 
   <td> <p>Immetti il codice HTML da convertire in Markdown.</p> </td> 
  </tr> 
  <tr> 
   <td role="rowheader">[!DNL GitHub Flavored Markdown] </td> 
   <td> <p>Abilitare questa opzione per convertire il HTML in [!DNL GitHub Flavored Markdown].</p> <p>Per ulteriori informazioni, vedere la scheda di riferimento Markdown nella documentazione di [!DNL GitHub].</p> </td> 
  </tr> 
 </tbody> 
</table>

## [!UICONTROL Markdown a HTML]

Questo modulo converte Markdown in HTML.

Durante la configurazione di questo modulo, vengono visualizzati i seguenti campi.

<table style="table-layout:auto"> 
 <col> 
 <col> 
 <tbody> 
  <tr> 
   <td role="rowheader">[!UICONTROL Markdown]</td> 
   <td> <p>Immetti il testo normale formattato per Markdown.</p> </td> 
  </tr> 
  <tr> 
   <td role="rowheader">[!DNL GitHub Flavored Markdown] </td> 
   <td> <p>Abilita questa opzione per convertire GitHub Flavored Markdown in HTML.</p> <p>Per ulteriori informazioni, vedere la scheda di riferimento Markdown nella documentazione di [!DNL GitHub].</p> </td> 
  </tr> 
  <tr> 
   <td role="rowheader">[!UICONTROL Sanitize]</td> 
   <td>Seleziona un’opzione per indicare se desideri rimuovere i tag HTML dal testo o dal HTML di escape.</td> 
  </tr> 
 </tbody> 
</table>
